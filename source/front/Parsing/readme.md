# 语法分析模块

Tastror updated in 2022.5.18

## 综述

语法分析采用 FIRST 标记符号的自顶向下分析法。因为 SysY 的语言大多一个 FIRST 符就够了，所以非常舒服。

| :warning:**WARNING**：仅仅在赋值语句中向后搜索了是否存在 `=`，在一个有大量表达式的数组赋值中，也许会影响性能。 |
| ------------------------------------------------------------ |

具体语法树的类型请查看 `define.h` 中 AST 部分。
